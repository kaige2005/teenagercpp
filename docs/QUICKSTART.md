# TeenC++ 快速上手指南

## 🎮 启动程序

双击 `TeenCppEdu.exe` 启动教学系统。

首次启动会自动创建数据库文件 `teen_cpp_student.db`。

## 📚 学习流程

### 第一步：进入课程
点击绿色关卡按钮（如"第1课：你好，C++！"）

### 第二步：阅读学习步骤
左侧显示本课的学习步骤：
```
📚 学习步骤
┌─────────────────────────────────┐
│ ● 步骤1: 认识代码结构            │
│   观察代码模板...                │
│   💡 点击看提示                  │
├─────────────────────────────────┤
│ ○ 步骤2: 修改输出内容            │
│   找到 cout 那一行...            │
├─────────────────────────────────┤
│ ○ 步骤3: 点击检查按钮            │
│   修改完成后，点击检查...        │
├─────────────────────────────────┤
│ 💡 温馨提示                      │
│ 1. 仔细阅读每个步骤              │
│ 2. 按提示修改代码                │
└─────────────────────────────────┘
```

### 第三步：修改代码
在右侧代码编辑器中根据步骤提示修改：
```cpp
// [你的代码在这里]
// 提示：修改双引号里面的文字
cout << "在这里写上你想说的话！" << endl;
```

### 第四步：检查代码
点击 **🔍 检查代码** 按钮

#### ✅ 检查通过
```
🎉 太棒了！通过了所有检查！得分：100/100

✓ 代码结构检查 (25/25分)
  代码结构正确！

✓ 基础语法检查 (25/25分)
  语法检查通过！

✓ cout 输出检查 (25/25分)
  找到 cout 输出语句！
...
```

#### ❌ 检查未通过
```
⚠️ 还有必要项没有通过：cout 输出检查，继续努力！

✗ cout 输出检查 (0/25分)
  还没有使用 cout 进行输出哦！

✓ 结构检查通过...

💡 提示：记得点击 "🔓 导师放行" 可以获得帮助！
```

### 第五步：生成项目
点击 **📂 生成项目** 按钮，系统会：
1. 在 `output/` 目录生成完整的 Dev-C++ 项目
2. 自动打开文件夹
3. 你可以用 Dev-C++ 打开并运行程序

### 第六步：通关解锁
通过检查后：
- 🎉 显示通关动画
- 💎 获得经验值（+100 EXP）
- 🏆 解锁下一关

## 💡 常见问题

**Q: 代码检查总是不通过怎么办？**
A: 仔细阅读步骤提示，也可以让导师点击"🔓 导师放行"按钮。

**Q: 生成的项目在哪里？**
A: 在程序目录的 `output/` 文件夹中，每个课程独立文件夹。

**Q: 可以用其他编辑器吗？**
A: 可以！生成项目后可以用 VS Code、Code::Blocks 等打开，但推荐使用 Dev-C++（教学要求）。

**Q: 进度保存在哪里？**
A: 保存在同目录的 `teen_cpp_student.db` 文件中，不要删除！
