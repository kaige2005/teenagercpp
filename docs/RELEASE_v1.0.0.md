## 🎉 发布概览

TeenC++ 教学系统首个正式版本发布！这是一款专为12-16岁青少年设计的游戏化C++编程学习平台，通过"勇者的C++冒险"故事背景，将编程学习转化为一场有趣的探险。

**核心亮点**:
- 🎮 游戏化界面，冒险地图展示学习进度
- ✍️ 内置代码编辑器，边学边练
- ✓ 智能代码检查，即时反馈学习成果
- 👨‍🏫 导师手动放行，灵活的教学控制
- 📦 一键生成 Dev-C++ 项目

---

## 🚀 功能详情

### 1. 游戏化学习界面

**冒险地图**
- 可视化关卡地图，每课呈现为一个冒险地点
- 已通关关卡绿色高亮显示
- 当前等级和经验值实时展示
- 点击关卡即可进入学习

**成就系统**
- 经验值(EXP)奖励机制
- 徽章系统（第1课获得"first_step"徽章）
- 等级系统（从Lv.1开始成长）

### 2. 课程学习系统

**第1课：你好，C++！**
- 3个学习步骤引导
- 每步配有详细说明和提示
- 右侧代码编辑器支持修改演练
- 预置代码模板降低门槛

**学习流程**
1. 认识代码结构 - 了解C++程序基本框架
2. 修改输出内容 - 修改cout语句体验输出
3. 检查代码 - 验证学习成果

### 3. 代码检查系统

**5维度检查**
| 检查类型 | 功能 | 示例 |
|----------|------|------|
| 语法检查 | 验证基本语法 | 括号匹配、分号缺失 |
| 关键字检查 | 检查必须使用的关键字 | 是否使用 cout、endl |
| 结构检查 | 验证代码结构 | main函数是否存在 |
| 输出检查 | 验证期望输出 | 输出内容是否匹配 |
| 风格检查 | 代码风格建议 | 缩进、命名规范 |

**灵活的检查模式**
- 通过检查：获得满分，自动解锁下一课
- 未通过检查：显示具体问题，仍可请求导师放行

### 4. 导师手动放行

**适用场景**
- 学生代码逻辑正确但检查器误判
- 学生有创意实现但不符合标准答案
- 教学灵活性需求

**操作步骤**
1. 代码检查未通过时，显示"🔓 导师放行"按钮
2. 点击后弹出放行对话框
3. 导师输入姓名和备注说明
4. 确认后学生通过该课程

**记录追溯**
- 所有放行记录保存到数据库
- 可追溯导师姓名、备注和时间

### 5. Dev-C++ 项目生成

**一键生成**
- 自动生成完整的Dev-C++项目
- 包含 `.dev` 项目文件
- 包含 `main.cpp` 源代码
- 包含 `Makefile.win` 构建配置

**使用方式**
1. 在学习界面点击"📂 生成项目"按钮
2. 项目生成到指定输出目录
3. 自动打开文件夹
4. 双击 `.dev` 文件在Dev-C++中打开
5. 编译并运行

## 📥 安装与使用

### 系统要求

**最低配置**
- 操作系统：Windows 10 (64位)
- 运行时：.NET Framework 4.8
- 内存：4GB
- 磁盘：100MB 可用空间

**推荐配置**
- 操作系统：Windows 10/11 (64位)
- 内存：8GB

### 安装步骤

**方式1：使用发布包**
1. 下载 `TeenCppEdu-v1.0.0.zip`
2. 解压到任意目录
3. 双击 `TeenCppEdu.exe` 运行

**方式2：从源码运行**
1. 确保已安装 .NET Framework 4.8
2. 克隆项目代码
3. 进入 `src/` 目录
4. 运行 `dotnet run --project TeenCppEdu`

### 首次使用

1. 运行程序，显示主界面
2. 点击"第1课：你好，C++！"
3. 按学习步骤阅读说明
4. 在代码编辑器中修改代码
5. 点击"🔍 检查代码"验证
6. 通过后自动解锁下一课
