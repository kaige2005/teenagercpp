## 一、核心方向调整

### 1.1 优先级重构

**原规划**: 快速开发 L04-L06，堆课程数量
**新规划**: ✅ **深度打磨体验，再扩展内容**

```
原计划: L1 → L2 → L3 → L4 → L5 → L6 (快速堆课)
新计划: L1 → L2 → L3 → ✅ 重构 L1/L2 → 打磨体验 → L4...
                    ↑
              三阶段新格式验证成功
```

**原因**: L03 三阶段新格式效果出色，内容更丰富、形式更生动

## 二、课程重构需求 (L01/L02)

### 2.1 重构目标

将 L01、L02 从**旧格式 (Steps)** 迁移到**新格式 (三阶段 Phase)**

| 课程 | 当前格式 | 目标格式 | 预估工作 |
|------|----------|----------|----------|
| L01 | 传统关卡 | 三阶段 (知识+实践+挑战) | 2周 |
| L02 | 传统关卡 | 三阶段 (知识+实践+挑战) | 2周 |

### 2.2 L01 重构要点

**课程主题**: "你好，C++！" → 保持主题，升级形式

**新结构设计**:
- **知识阶段** (50XP)
  - 场景: 程序员的第一个魔法咒语
  - 概念: 程序结构、main函数、输出语句
  - 测验: 3道选择题
  - 填空: 3道代码填空
  
- **实践阶段** (100XP)
  - 任务: 编写个性化问候程序
  - 模板: 含TODO标记的main.cpp
  - 检查: 5项规则
  
- **挑战阶段** (20XP)
  - Bug猎手: 引号不匹配、分号缺失等典型错误
  - 徽章: hello_bug_hunter

### 2.3 L02 重构要点

**课程主题**: "分支魔法" → 保持主题，升级形式

**新结构设计**:
- **知识阶段** (60XP)
  - 场景: 猜数字游戏的智慧之门
  - 概念: if/else、比较运算符、逻辑判断
  - 测验: 3道选择题
  - 填空: 4道代码填空
  
- **实践阶段** (100XP)
  - 任务: 完整猜数字游戏
  - 模板: 含随机数和输入的框架
  - 检查: 4项规则 (已有，可复用)
  
- **挑战阶段** (30XP)
  - Bug猎手: 逻辑错误、边界条件
  - 徽章: branch_bug_hunter

## 三、UI/UX 优化需求清单

### 3.1 布局问题

#### 问题 1: 文字遮挡/需要滑动
**现象**: 某些面板文字被遮挡，需要用户手动滚动
**具体位置**: 待确认 (知识阶段? 实践阶段?)
**期望**: 自适应布局，核心内容一屏可见

#### 问题 2: 完成的课程显示问题
**现象**: L02 通关后回到主界面，按钮仍是灰色
**可能原因**: 
- 解锁状态刷新逻辑问题
- 主界面地图状态未正确更新
- XP 累积但没有正确显示进度

**期望**: 通关后按钮变绿色，显示 "已完成"

#### 问题 3: 按钮位置偏下，被遮挡
**现象**: 底部按钮显示不全
**具体位置**: 课程窗体底部 (检查代码/生成项目按钮?)
**可能原因**: 
- 窗体高度计算问题
- 分辨率适配问题
- 按钮容器 Dock 属性设置

**期望**: 所有按钮完整可见，不被任务栏/边框遮挡

#### 问题 4: 整体布局偏右上
**现象**: 界面元素整体偏向右上方
**可能原因**:
- 主面板 Padding/Margin 设置
- 自适应布局算法问题
- 固定坐标 vs 相对坐标

**期望**: 内容居中或均匀分布，视觉平衡

## 四、经验值系统优化

### 4.1 当前问题

**现状**: 
- L01: 100 XP
- L02: 150 XP
- L03: 知识60 + 实践100 + 挑战30 = 190 XP

**问题**: 
- XP 计算规则不够清晰
- 等级提升感知不强
- 缺乏 XP 使用/消耗场景

### 4.2 构思方向 (待深入设计)

#### 方向 A: 技能树系统
```
经验值 → 解锁技能
├── 代码高亮
├── 自动补全提示
├── 调试工具
└── 主题皮肤
```

#### 方向 B: 成就徽章体系
```
闯关徽章: 每课通关
技能徽章: 语法/逻辑/算法
连续徽章: 连续学习天数
探索徽章: 发现隐藏内容
```

#### 方向 C: 等级特权
```
Lv.1 新手: 基础功能
Lv.5 学徒: 解锁挑战模式
Lv.10 法师: 自定义主题
Lv.20 大师: 解锁隐藏课程
```

#### 方向 D: XP 消耗场景
```
- 复活/提示消耗 XP
- 兑换皮肤/主题
- 解锁额外练习题
```

**待决策**: 哪种方向更适合青少年学习?
