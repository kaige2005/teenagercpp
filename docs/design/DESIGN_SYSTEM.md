# TeenC++ 设计系统 v1.0

> 为青少年 C++ 编程教育平台建立的视觉和交互规范

---

## 1. 设计原则

### 1.1 核心愿景

**让编程学习像探险游戏一样有趣**

- 🎮 **游戏化**: 用探险、魔法、挑战等元素包装技术概念
- 🎯 **成就感**: 即时反馈、进度可见、徽章奖励
- 🧠 **渐进式**: 从简单到复杂， scaffolding 学习
- 💡 **清晰性**: 复杂概念用可视化、故事化表达

### 1.2 用户画像

**主要用户**: 12-16岁中国青少年

| 属性 | 描述 |
|------|------|
| **认知水平** | 逻辑思维发展期，需要具体例子 |
| **注意力** | 15-20分钟集中，需要变化刺激 |
| **动机** | 游戏化驱动、成就收集、社交分享 |
| **技术背景** | 零编程基础，第一次接触代码 |
| **痛点** | 语法枯燥、错误挫败、缺乏即时反馈 |

### 1.3 设计价值观

```
趣味 > 专业
渐进 > 全面
反馈 > 讲解
探索 > 灌输
```

## 2. 色彩系统

### 2.1 主色调

| 颜色 | Hex | RGB | 用途 |
|------|-----|-----|------|
| **深蓝** | `#2D3447` | rgb(45,52,71) | 主背景 |
| **暗蓝** | `#232938` | rgb(35,41,56) | 次级背景 |
| **金色** | `#FFC850` | rgb(255,200,80) | 强调色、成就 |
| **绿色** | `#64C880` | rgb(100,200,128) | 成功、通过 |
| **红色** | `#FF6464` | rgb(255,100,100) | 错误、危险 |

### 2.2 功能色

| 颜色 | Hex | 用途 |
|------|-----|------|
| **知识蓝** | `#5090E0` | 知识阶段标识 |
| **实践橙** | `#F09040` | 实践阶段标识 |
| **挑战紫** | `#B050E0` | 挑战阶段标识 |
| **锁定灰** | `#606870` | 未解锁内容 |

### 2.3 文本色

| 颜色 | Hex | 用途 |
|------|-----|------|
| **主文字** | `#F0F0F0` | 主要文本 |
| **次级文字** | `#B0B8C0` | 辅助说明 |
| **弱化文字** | `#708090` | 禁用、提示 |

### 2.4 色彩使用规则

```
背景层次:
- 最底层: #1A1F2A (窗口背景)
- 主背景: #2D3447 (面板背景)
- 次级背景: #383F50 (卡片、输入框)
- 最上层: #404858 (按钮、激活状态)

强调使用:
- 金色: 主要按钮、成就、标题
- 绿色: 通过、成功、正向反馈
- 红色: 错误、删除、警告

阶段配色:
- 知识阶段: 蓝色系 (#5090E0)
- 实践阶段: 橙色系 (#F09040)  
- 挑战阶段: 紫色系 (#B050E0)
```

## 4. 间距与布局

### 4.1 间距标准

```
基础单位: 8px

小间距: 8px  (元素之间)
中间距: 16px (段落之间)
大间距: 24px (面板之间)
超大间距: 32px (区域之间)
```

### 4.2 面板尺寸

| 面板 | 宽度 | 高度 | 说明 |
|------|------|------|------|
| **课程窗体** | 1200 | 800 | 默认尺寸 |
| **左侧步骤区** | 350 | 自适应 | DockStyle.Left |
| **右侧编辑区** | 850 | 自适应 | DockStyle.Fill |
| **步骤面板** | 320 | 400 | 固定大小 |

```
按钮间距: 8px
标签边距: 12px
面板内边距: 16px

```

### 4.4 响应式断点

```
小屏: < 1366 (笔记本)
中屏: 1366-1920 (标准)
大屏: > 1920 (高分屏)

最小支持: 1280x720
最佳效果: 1920x1080
```

## 7. 动画与反馈

### 7.1 过渡动画

```css
/* 按钮悬停 */
transition: background-color 0.2s ease;

/* 面板切换 */
transition: opacity 0.3s ease, transform 0.3s ease;

/* 进度条 */
transition: width 0.5s ease-out;
```

### 7.2 反馈时机

| 行为 | 反馈 | 时机 |
|------|------|------|
| 悬停 | 颜色变化 | 100ms |
| 点击 | 轻微缩放 | 50ms |
| 成功 | 绿色闪烁 | 300ms |
| 错误 | 红色抖动 | 300ms |
| 完成 | 金色光芒 | 500ms |
