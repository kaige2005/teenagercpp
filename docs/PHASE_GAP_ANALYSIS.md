## 流程使用建议

### 当前项目（v1.0收尾）

由于项目已基本开发完成，建议：

1. **立即执行**: P0任务（测试+发布文档）
2. **发布v1.0**: 完成P0后即可标记v1.0
3. **进入v1.1迭代**: 严格按照新流程执行

### 后续迭代（v1.1开始）

从v1.1开始，所有新功能必须遵循以下流程：

```
新需求 → PRD模板 → 设计评审 → 开发 → 代码审查 → 测试 → 发布
               ↑             ↑           ↑
            [评审会]      [检查清单]   [质量门禁]
```

**v1.1建议功能**:
- 第3-4周课程内容
- 单元测试全覆盖
- 代码挑战模式（"Bug猎手"模式）
