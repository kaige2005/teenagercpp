# TeenC++ v1.2.0 测试用例集

## 一、功能测试用例

### FT-001: 课程数据加载
| 项目 | 内容 |
|------|------|
| 前置条件 | 应用已启动，数据库初始化完成 |
| 步骤 | 1. 点击主界面课程按钮 |
| 预期结果 | 课程窗体正常加载，无异常抛出 |
| 数据变异 | L01旧格式, L02旧格式, L03新格式 |

### FT-002: 知识阶段完整流程 (L03)
| 项目 | 内容 |
|------|------|
| 前置条件 | 进入L03课程 |
| 步骤 | 1. 点击"开始学习"<br>2. 浏览场景故事<br>3. 阅读概念讲解<br>4. 完成测验(3题)<br>5. 完成代码填空 |
| 预期结果 | 每环节XP累计正确，总XP=60 |
| 检查点 | 进度显示正确，可正常进入实践阶段 |

### FT-003: 代码编辑器显示
| 项目 | 内容 |
|------|------|
| 前置条件 | 进入实践阶段 |
| 步骤 | 观察右侧代码编辑区域 |
| 预期结果 | RichTextBox可见，显示模板代码 |
| 检查点 | 有滚动条、可编辑、语法高亮(如有) |

### FT-004: Bug猎手挑战
| 项目 | 内容 |
|------|------|
| 前置条件 | L03实践阶段完成 |
| 步骤 | 1. 点击Bug猎手<br>2. 查看Bug代码<br>3. 点击问题行<br>4. 输入修复代码 |
| 预期结果 | 正确识别Bug+修复=30XP |
| 数据变异 | 正确修复, 错误修复, 跳过挑战 |

## 二、回归测试用例

### RT-001: L01完整通关 (旧格式兼容)
```
步骤：
1. 启动应用
2. 点击"第1课：你好，C++！"
3. 验证：显示代码编辑器
4. 修改cout语句
5. 点击"检查代码"
6. 验证：显示通过信息
7. 点击"生成项目"
8. 验证：Dev-C++项目生成成功
9. 关闭课程窗体
10. 验证：主界面XP增加

预期：全流程无异常
```
