# TeenC++ 青少年C++编程教学系统

> 🎮 让C++学习像游戏冒险一样有趣！

## 项目概述

这是一个专为 **12-16岁青少年** 设计的游戏化C++编程学习平台。系统通过 **故事化课程**、**关卡式教学**、**代码模板生成** 和 **智能评分验证**，帮助学生系统掌握C++编程基础。

本系统特别设计为 **"内容管理系统"** 形态，与 **Dev-C++** 配合使用：
- 学生在系统中学习课程内容
- 系统在本地生成代码练习模板
- 学生在 Dev-C++ 中编写、运行代码
- 学生提交代码文件和运行结果，系统验证评分

## 核心特性

### 🎯 游戏化学习体验
- **8周冒险剧情**：勇者的C++大陆冒险故事
- **徽章解锁**：完成每周课程获得勇者徽章
- **积分系统**：任务/作业/项目多维评分
- **代码展示墙**：记录每一步成长

### 📚 内容管理系统
- 结构化课程编排（周 → 课时 → 练习）
- 富文本知识点讲解
- 填空式代码模板生成
- 项目模板与里程碑管理

### ✅ 智能练习验证
- 代码结构检查（关键字、完整性）
- 运行结果比对（精确/模糊匹配）
- 多维度评分反馈
- 导师人工确认机制

## 技术栈

- **开发语言**: C# (.NET Framework 4.6.1+)
- **UI框架**: Windows Forms
- **数据存储**: SQLite (本地单用户)
- **目标平台**: Windows 7/10/11

## 快速开始

### 环境要求
- Windows 7 SP1 或更高版本
- .NET Framework 4.6.1 或更高版本
- Dev-C++ 5.11 便携版（用于代码编写运行）

### 安装步骤

1. 下载最新版本 `TeenCppEdu.zip`
2. 解压到任意目录（如 `D:\TeenCppEdu`）
3. 运行 `TeenCppEdu.exe`
4. 首次启动自动创建本地数据库和学习空间

### 使用流程

```
1. 启动系统 → 看到课程地图
2. 选择课时 → 学习故事背景和知识点
3. 点击"开始练习" → 系统自动生成代码模板
4. 在 Dev-C++ 中打开模板文件 → 编写代码 → 运行
5. 返回系统 → 提交代码文件和运行结果
6. 查看评分 → 通过解锁下一课
7. 完成本周所有课时 → 挑战周项目
8. 周项目通过 → 获得周章徽章
```

## 课程大纲（试点版：第1-2周）

### 第1周：勇者启程·初识C++大陆
- **第1课** - 魔法咒语格式：C++基础结构、cout输出
- **第2课** - 勇者背包：变量定义、数据类型、cin输入
- **第3课** - 伤害计算：算术运算符
- **周项目** - 猜数字游戏（基础版）
- **徽章** - 🛡️ 新手村勇者

### 第2周：分支魔法·选择的力量
- **第1课** - 岔路口的选择：if/else双分支、关系运算符
- **第2课** - 多路迷宫：if/else if/else多分支、分支嵌套
- **第3课** - 多重机关门：逻辑运算符 && || !
- **周项目** - 猜数字游戏（进阶版）
- **徽章** - 🔍 迷雾探索者
