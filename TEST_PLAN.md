## 编译步骤

### 步骤1: 打开解决方案
双击 `src\TeenCppEdu.sln`

### 步骤2: 检查引用状态
- 右键解决方案 → "还原 NuGet 包"
- 确认没有黄色叹号图标

### 步骤3: 编译
- 菜单: 生成 → 生成解决方案
- 或使用快捷键: Ctrl+Shift+B

### 步骤4: 处理编译错误
如果报错，检查：
1. .NET Framework 4.8 是否已安装
2. NuGet 包是否正确恢复
3. 所有 using 语句是否正确

## 常见问题排查

### 编译错误
| 错误信息 | 可能原因 | 解决 |
|----------|----------|------|
| 找不到 Newtonsoft.Json | NuGet 包未恢复 | 右键解决方案 → 还原 NuGet 包 |
| 找不到 System.Data.SQLite | 同上 | 同上 |
| 语法错误 CS8370 | 语言版本问题 | 确认 LangVersion 为 8.0 |
| 类型或命名空间不存在 | using 错误 | 检查命名空间引用 |

### 运行时错误
| 错误信息 | 可能原因 | 解决 |
|----------|----------|------|
| 找不到 courses/lesson01 | 课程数据未复制 | 将 courses 目录复制到输出目录 |
| 数据库锁定 | SQLite 并发问题 | 关闭程序重新打开 |
| 生成项目失败 | 路径权限问题 | 检查 output 目录是否有写入权限 |
