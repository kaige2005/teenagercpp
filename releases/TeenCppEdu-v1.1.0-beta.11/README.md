## 🚀 快速开始

### 系统要求
- Windows 10/11 (64位)
- .NET Framework 4.8
- Dev-C++ 5.11 (用于代码练习)

### 运行方式
双击 `TeenCppEdu.exe` 启动程序

### 数据存储
用户学习进度保存在运行目录的 `teen_cpp_student.db` 文件中。
