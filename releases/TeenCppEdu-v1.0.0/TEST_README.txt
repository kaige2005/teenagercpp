TeenC++ v1.0.0 诊断版本
====================

此版本包含详细的错误诊断和日志记录。

## 测试步骤

1. 运行 TeenCppEdu.exe
2. 点击「第1课：你好，C++！」
3. 观察结果：
   - 如果成功：课程窗口会打开
   - 如果失败：会弹出错误对话框，包含详细错误信息和日志路径

## 查看日志

日志文件位置：
%AppData%\TeenCppEdu\logs\TeenCppEdu_YYYYMMDD.log

在文件资源管理器地址栏直接输入：
%AppData%\TeenCppEdu\logs\

## 反馈信息

如果无法进入课程，请提供：
1. 错误对话框截图（如果有）
2. 日志文件内容
3. 操作系统版本
4. 程序运行路径

## 文件结构检查

确认以下目录存在：
courses\l01\
courses\l01\lesson.json
courses\l01\checks\rules.json
courses\l01\templates\main.cpp
