## 新功能

### 🎓 第3课：数组探险
- **三阶段渐进式学习**:
  1. **知识讲解** (60XP): 通过故事理解数组概念
  2. **编程实践** (100XP): 完成数组求和编程任务
  3. **Bug猎手** (30XP): 找出并修复隐藏的Bug

### 🎮 新格式课程系统
- 支持知识/实践/挑战三阶段
- 阶段进度自动保存
- 断点续学功能

### 🐛 Bug猎手模式
- 交互式代码Bug修复挑战
- 即时反馈修复结果
- 详细解释正确答案
