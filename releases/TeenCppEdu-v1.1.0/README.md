## 🚀 快速开始

### 系统要求
- Windows 10/11 (64位)
- .NET Framework 4.8
- Dev-C++ 5.11 (推荐，用于代码练习)

### 安装步骤
1. 解压 `TeenCppEdu-v1.1.0.zip` 到任意目录
2. 双击 `TeenCppEdu.exe` 启动

### 数据备份
学习进度保存在运行目录的 `teen_cpp_student.db` 文件中，建议定期备份。
