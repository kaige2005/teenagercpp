#### BUG-20260322-004: 课程解锁后按钮状态不同步

## [1.0.0] - 2026-03-21

### 🎉 首个正式版本发布

TeenC++ 教学系统 v1.0.0 正式版发布！这是面向12-16岁青少年的游戏化C++编程学习平台首个可用版本。

### 🚀 新增功能

#### 核心学习系统
- **游戏化主界面** - 冒险地图可视化展示学习进度，等级徽章系统
- **课程学习系统** - 分步式教学，左侧引导步骤，右侧代码编辑
- **第1课完整内容** - "你好，C++！" 零基础入门第一课

#### 代码检查系统
- **实时代码检查** - 支持学生编写代码后的即时检查
- **5种检查器**:
  - 语法检查器 (SyntaxChecker) - 验证基本语法正确性
  - 关键字检查器 (KeywordChecker) - 检查必须使用的C++关键字
  - 结构检查器 (StructureChecker) - 验证代码结构完整性
  - 输出检查器 (OutputChecker) - 验证预期输出匹配
  - 风格检查器 (StyleChecker) - 代码风格建议
- **通过/失败反馈** - 明确显示检查结果和得分

#### 导师功能
- **灵活检查模式** - 检查仅作参考，不强制阻塞学习进度
- **导师手动放行** - 支持导师人工审核并放行不通过的代码
- **放行记录** - 记录导师姓名、备注和放行时间

#### 项目生成
- **Dev-C++项目生成** - 一键生成完整的Dev-C++项目(.dev + .cpp)
- **自动打开文件夹** - 生成后自动打开资源管理器
- **即用即编译** - 生成的项目可直接在Dev-C++中编译运行

#### 数据管理
- **SQLite持久化** - 学习进度自动保存
- **关卡解锁机制** - 通关后自动解锁下一课
- **经验值系统** - 完成课程获得EXP奖励

### 📦 技术实现

- C# WinForms + .NET Framework 4.8
- SQLite 数据存储
- Newtonsoft.Json 配置管理
- 模块化架构设计 (Core/Services/UI)

### 🎯 系统要求

- Windows 10/11 (64位)
- .NET Framework 4.8
- Dev-C++ 5.11 (用于代码练习)

## 如何升级

### 从源码升级
1. 备份 `data/` 目录（包含学习进度）
2. 拉取新版本代码
3. 重新编译
4. 将备份数据复制回 `data/` 目录

### 从发布包升级
1. 备份当前版本的 `data/` 目录
2. 解压新版本覆盖
3. 恢复 `data/` 目录
